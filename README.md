# Quran Tafsir PWA

تطبيق قرآن وتفسير يعمل بالكامل من الملفات المحلية (بدون Backend).

## التشغيل المحلي

- افتح المشروع عبر أي static server (مثل Live Server).
- أول زيارة للموقع: سيتم تحميل الملفات وتخزينها في Cache عبر Service Worker.
- بعد ذلك يعمل التطبيق Offline.

## PWA

- `manifest.json` يحتوي الأيقونات والألوان وإعدادات التثبيت.
- `sw.js` يقوم بتخزين ملفات التطبيق والبيانات للاستعمال دون إنترنت.

## ملاحظة البيانات

الملفات الموجودة داخل `data/` حالياً عبارة عن عينة صغيرة للتجربة.

لاستخدام التطبيق الكامل:

1. استبدل `data/quran.json` بالنسخة الكاملة.
2. استخدم ملف تفسير الصفحات `data/tafsir_page.json`.
3. استبدل `data/tafsir_page.json` بالنسخة الكاملة (604 صفحة).

## توليد التفسير (مرة واحدة للمطور)

المتطلبات:

- Node.js 18+ (لوجود `fetch` المدمج)
- مفتاح Gemini في المتغير `GEMINI_API_KEY`

تشغيل:

```bash
set GEMINI_API_KEY=YOUR_KEY
لا يوجد توليد آلي للتفسير داخل المشروع.
```

السكريبت:

- يجلب نص القرآن من API.
- يعتمد تفسير الصفحات على `data/tafsir_page.json`.
- يولد تفسير صفحات في `data/tafsir_page.json`.
- يكمل تلقائياً من آخر نقطة محفوظة.
- ينام `500ms` بين الطلبات.
- يطبع التقدم بالشكل: `✅ آية 450/6236` و `✅ صفحة 120/604`.
